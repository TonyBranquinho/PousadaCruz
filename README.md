# 🌊 Pousada Cruz — Website Institucional (Projeto Comercial)
👉 https://tonybranquinho.github.io/PousadaCruz/  
👉 https://PousadaCruz.com.br/

Website institucional desenvolvido sob demanda para o **Hotel Pousada Cruz**, localizada na Praia do Ervino, em São Francisco do Sul (SC).

Projeto com foco em:

- Presença digital profissional
- SEO local
- Conversão via WhatsApp
- Experiência visual moderna
- Responsividade completa

---

## 🎯 Objetivo do Projeto

Criar uma presença online profissional para a pousada, com foco em:

- Aumentar visibilidade no Google
- Facilitar contato direto via WhatsApp
- Melhorar credibilidade da marca
- Exibir acomodações e localização de forma clara
- Fortalecer posicionamento local

---

## 🧱 Stack Utilizada

- **HTML5 semântico**
- **CSS3 (Flexbox, Media Queries, animações)**
- **JavaScript**
- **Google Maps Embed**
- **Google Fonts**
- **Schema.org (JSON-LD – LodgingBusiness)**

---

## 🔎 Estratégia de SEO Local

O projeto inclui implementação de:

- Meta description otimizada
- Open Graph (compartilhamento em redes sociais)
- Dados estruturados (`@type: LodgingBusiness`)
- Geolocalização com latitude e longitude
- Palavras-chave estratégicas:
  - “Pousada na Praia do Ervino”
  - “São Francisco do Sul SC”
  - “Pousada à beira-mar”

Estrutura preparada para indexação orgânica no Google.
